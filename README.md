## 프로젝트 설명

NestJS 서버 사이드 렌더링을 사용하여 MVC 패턴의 실시간 웹 애플리케이션입니다.
소켓 프로그래밍, 이벤트 설계, MVC 디자인 패턴 등 다양한 개념을 익히기 위한 study 토이 프로젝트 입니다.

## Installation

```bash
$ npm i
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# debug mode
$ npm run start:debug

# production mode
$ npm run start:prod
```

## Build

```bash
$ npm run build
```
